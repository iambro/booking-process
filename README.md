Project is created with:
* HTML5
* CSS3 (SASS)
* JavaScript
* React

## Instalation
* Git clone https://github.com/iambro/booking-process.git to local repository.
* Install node.js https://nodejs.org/en/download/
* Open folder with an app in console (cd booking-process)
* npm install
* npm start
